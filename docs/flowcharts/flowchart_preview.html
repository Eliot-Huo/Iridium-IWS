3️⃣ 關鍵要點

📋 操作前提條件

變更費率 (update_subscriber_plan)

  • ✅ 設備屬於您的 Device Pool
  • ✅ 設備狀態為 ACTIVE
  • ✅ 有 subscriberAccountNumber

暫停設備 (suspend_subscriber)

  • ✅ 設備屬於您的 Device Pool
  • ✅ 設備狀態為 ACTIVE
  • ❌ RESERVED 或 SUSPENDED 無法暫停

註銷設備 (deactivate_subscriber)

  • ✅ 設備屬於您的 Device Pool
  • ✅ 設備狀態為 ACTIVE 或 SUSPENDED
  • ✅ 註銷後立即釋放，恢復為 RESERVED
  • ✅ 可用於準備測試設備

🔍 常見錯誤

  • "Account not found for IMEI"
    → 設備處於 RESERVED 狀態（未啟動）或不存在
  • "Device does not belong to account"
    → 設備不在您的 Device Pool
  • "Invalid state for device"
    → 設備狀態不符合操作要求

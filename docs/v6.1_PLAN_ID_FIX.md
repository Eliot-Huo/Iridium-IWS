## 🔧 v6.1 修正內容

### **問題診斷**

activateSubscriber 錯誤：
```
The Long value supplied [SBD12] is not valid.
```

**原因**：`sbdBundleId` 欄位必須是 Long 型別（純數字字串），不能包含字母。

### **解決方案**

#### **1. Plan ID 自動轉換**

新增 `_extract_plan_id_digits` 方法：

```python
def _extract_plan_id_digits(self, plan_id: str) -> str:
    """
    提取 plan_id 中的純數字
    
    範例：
    - "SBD12" → "12"
    - "SBDO" → "0"
    - "SBD17" → "17"
    - "12" → "12"
    """
    # 移除所有非數字字元
    digits = re.sub(r'\D', '', plan_id)
    
    # 如果沒有數字，預設為 "0"
    if not digits:
        digits = "0"
    
    print(f"[IWS] Plan ID conversion: '{plan_id}' → '{digits}'")
    
    return digits
```

**在 activate_subscriber 中自動應用**：

```python
# 關鍵：提取純數字
plan_id_digits = self._extract_plan_id_digits(plan_id)

body = f'''...
    <sbdBundleId>{plan_id_digits}</sbdBundleId>
...'''
```

#### **2. 新增 getSBDBundles 方法**

查詢可用的 SBD 方案：

```python
def get_sbd_bundles(self, model_id: Optional[str] = None) -> Dict:

```

**使用範例**：



**關鍵**：Action 名稱必須準確參與簽章計算：

```python
action_name = 'getSBDBundles'

message = f"{action_name}{timestamp}"
```

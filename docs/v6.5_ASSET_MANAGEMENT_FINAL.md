### **2. 安全性修正 - 刪除所有 Hardcode** ⭐⭐⭐

**刪除的 Hardcoded 資訊**：
- ❌ 程式碼中的明文帳號
- ❌ 程式碼中的明文密碼
- ❌ 程式碼中的明文 SP 帳號
- ❌ 註解中的敏感資訊
- ❌ Docstring 中的敏感資訊
- ❌ `__init__` 預設值中的明文

**修正前（v6.4）**：
```python
def __init__(self, 
             username: Optional[str] = None,
             password: Optional[str] = None,
             sp_account: Optional[str] = None,
             ...):
    """
    Args:
        username: IWS 使用者名稱 (預設: IWSN3D)  # ← 不應出現
        password: IWS Secret Key (預設: FvGr2({sE4V4TJ:)  # ← 不應出現
        sp_account: Service Provider Account Number (預設: 200883)  # ← 不應出現
    """
```

**修正後（v6.5）**：
```python
def __init__(self, 
             username: Optional[str] = None,
             password: Optional[str] = None,
             sp_account: Optional[str] = None,
             ...):
    """
    Args:
        username: IWS 使用者名稱（從 settings 匯入）
        password: IWS Secret Key（從 settings 匯入）
        sp_account: Service Provider Account Number（從 settings 匯入）
    """
```

**安全性原則**：
- ✅ 所有憑證從 `config.settings` 匯入
- ✅ 程式碼中不含任何明文帳密
- ✅ 註解和文件中不含敏感資訊

### **3. 布林值改為數字 - 解決 500 錯誤** ⭐⭐⭐

**問題**：IWS API 不接受 `[false]` 字串，導致 500 錯誤。

**解決方案**：布林值一律轉換為數字。

**新增方法**：
```python
def _bool_to_int(self, value: bool) -> str:
    """
    將布林值轉換為數字字串
    
    Args:
        value: 布林值
        
    Returns:
        str: "1" (True) 或 "0" (False)
    """
    return "1" if value else "0"
```

**參數型別變更**：
```python
# v6.4（錯誤）
def update_subscriber_plan(self, ..., demo_and_trial: str = 'false'):
    ...
    <demoAndTrial>false</demoAndTrial>  # ← IWS 不接受

# v6.5（正確）
def update_subscriber_plan(self, ..., demo_and_trial: bool = False):
    demo_and_trial_value = self._bool_to_int(demo_and_trial)
    ...
    <demoAndTrial>0</demoAndTrial>  # ← 使用數字 ✓
```

**影響的標籤**：
- `<demoAndTrial>` → 使用 `0` 或 `1`

## 💡 **关键发现**

### **1. WSDL 文档 vs 实际 API**

**WSDL 可能说**：
- `subscriberAccountNumber`
- `<plan>` 对象包裹参数

**实际 API 要求**：
- `accountNumber`（在 `<subscriber>` 列表中）
- 参数直接放在 `<request>` 下

**教训**：
- ✅ WSDL 是指导，不是绝对真理
- ✅ 实际 API 测试才是最终验证
- ✅ 字段名和结构可能与文档不同

---

### **2. accountSearch 返回结构**

**不是单个结果**：
```xml
<subscriber>...</subscriber>  ← 单个
```

**而是列表**：
```xml
<subscribers>
    <subscriber>...</subscriber>  ← 需要遍历
    <subscriber>...</subscriber>
    <subscriber>...</subscriber>
    ...
</subscribers>
```

**需要**：
1. 找到所有 `<subscriber>` 元素
2. 遍历每个订阅者
3. 匹配 `<imei>` 字段
4. 返回对应的 `<accountNumber>`

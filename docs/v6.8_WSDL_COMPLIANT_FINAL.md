## 🔧 v6.8 重大修正（5 項）

### **修正 1: getSBDBundles - 使用 Plan 對象** ⭐⭐⭐

**錯誤（v6.7）**：
```xml
<tns:getSBDBundles>
    <request>
        ...
        <serviceType>SHORT_BURST_DATA</serviceType>  <!-- 錯誤！ -->
    </request>
</tns:getSBDBundles>
```

**正確（v6.8 - 根據 WSDL p.161-162）**：
```xml
<tns:getSBDBundles>
    <request>
        <iwsUsername>IWSN3D</iwsUsername>
        <signature>...</signature>
        <serviceProviderAccountNumber>200883</serviceProviderAccountNumber>
        <timestamp>...</timestamp>
        <plan>
            <fromBundleId>0</fromBundleId>  <!-- 必填 -->
            <forActivate>true</forActivate>  <!-- 必填 -->
            <modelId>...</modelId>  <!-- 可選 -->
        </plan>
    </request>
</tns:getSBDBundles>
```

**欄位說明**：
- `fromBundleId`：新啟用用 "0"，更新現有用戶用現有 bundle ID
- `forActivate`："true" (新啟用) 或 "false" (更新現有)
- `modelId`：可選，用於特殊設備型號（如 EDGE Pro）

### **修正 3: SBD Plan 結構 - 移除不存在的欄位** ⭐⭐⭐

**錯誤（v6.7）**：
```xml
<plan>
    <lritFlagstate>0</lritFlagstate>  <!-- 錯誤型別！ -->

```

**正確（v6.8 - 根據 WSDL p.286）**：
```xml
<plan>
    <sbdBundleId>17</sbdBundleId>  <!-- Long 型別 ✓ -->
    <lritFlagstate></lritFlagstate>  <!-- String：3字元或空字串 ✓ -->
```

- ✅ `ringAlertsFlag`：Boolean 型別（"true" 或 "false"）

### **修正 5: Boolean 格式 - 使用 "true"/"false"** ⭐⭐

**錯誤（v6.7）**：
```python
def _bool_to_int(self, value: bool) -> str:
    return "1" if value else "0"  # 錯誤！
```

**正確（v6.8）**：
```python
def _bool_to_string(self, value: bool) -> str:
    return "true" if value else "false"  # 正確 ✓
```

**應用**：
- `forActivate`："true" 或 "false"
- `ringAlertsFlag`："true" 或 "false"
- 所有 Boolean 欄位都使用字串，不使用數字

## 🚀 API 使用範例

### **查詢可用方案**

```python
from src.infrastructure.iws_gateway import IWSGateway

gateway = IWSGateway()

# 查詢新啟用的方案
result = gateway.get_sbd_bundles(
    from_bundle_id="0",      # 新啟用用 "0"
    for_activate=True,        # 是新啟用
    model_id=None             # 可選
)

print(f"找到 {result['count']} 個方案")
for bundle in result['bundles']:
    print(f"Bundle ID: {bundle.get('id')}, Name: {bundle.get('name')}")
```

---

### **變更費率**

```python
# 變更設備費率
result = gateway.update_subscriber_plan(
    imei='300534066711380',
    new_plan_id='17',         # 或 'SBD17'（自動轉換）
    lrit_flagstate='',        # 空字串（非 LRIT）
    ring_alerts_flag=False    # Boolean
)

print(f"Transaction ID: {result['transaction_id']}")
```

## 📝 關鍵學習

### **1. WSDL 是唯一真相來源**
- 不能假設方法名稱或欄位
- 必須完全按照官方文件實作

### **2. 物件結構很重要**
- getSBDBundles 需要 Plan 對象，不是簡單參數
- accountUpdate 需要完整的帳戶對象

### **3. 型別必須正確**
- Boolean：字串 "true"/"false"
- Long：純數字字串
- String：依欄位定義

### **4. 方法重用**
- setSubscriberAccountStatus 用於暫停、恢復、註銷
- 不需要單獨的 deactivateSubscriber 方法

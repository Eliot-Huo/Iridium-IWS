## ❌ 發現的重大錯誤（5 個）

### **錯誤 1: getSBDBundles 方法完全錯誤** ⭐⭐⭐

**v6.7 實作（錯誤）**：
```xml
<tns:getSBDBundles>
    <request>
        <iwsUsername>IWSN3D</iwsUsername>
        <signature>...</signature>
        <serviceProviderAccountNumber>200883</serviceProviderAccountNumber>
        <timestamp>...</timestamp>
        <serviceType>SHORT_BURST_DATA</serviceType>  <!-- 錯誤！ -->
    </request>
</tns:getSBDBundles>
```

**正確實作（根據 WSDL p.161-162）**：
```xml
<tns:getSBDBundles>
    <request>
        <iwsUsername>IWSN3D</iwsUsername>
        <signature>...</signature>
        <serviceProviderAccountNumber>200883</serviceProviderAccountNumber>
        <timestamp>...</timestamp>
        <plan>  <!-- 必填！ -->
            <fromBundleId>0</fromBundleId>  <!-- 必填：新啟用用 0 -->
            <forActivate>true</forActivate>  <!-- 必填：是否為新啟用 -->
            <modelId>...</modelId>  <!-- 可選：設備型號 -->
        </plan>
    </request>
</tns:getSBDBundles>
```

**說明**：
- getSBDBundles 需要 **Plan 對象**，不是 serviceType
- fromBundleId：新啟用用 "0"，更新時用現有 bundle ID
- forActivate："true" (新啟用) 或 "false" (現有用戶)

### **錯誤 3: SBD Plan 結構完全錯誤** ⭐⭐⭐

**v6.7 實作（錯誤）**：
```xml
<plan>
    <lritFlagstate>0</lritFlagstate>
    <demoAndTrial>0</demoAndTrial>  <!-- 此欄位不存在！ -->
    <sbdBundleId>17</sbdBundleId>
    <ringAlertsFlag>0</ringAlertsFlag>
</plan>
```

**正確實作（根據 WSDL p.286）**：
```xml
<plan>
    <sbdBundleId>17</sbdBundleId>  <!-- 必填：Long 型別 -->
    <lritFlagstate></lritFlagstate>  <!-- 必填：3字元或空字串 -->
    <ringAlertsFlag>false</ringAlertsFlag>  <!-- 可選：Boolean -->
</plan>
```

**關鍵發現**：
- **沒有 demoAndTrial 欄位**！這個欄位不存在於 sbdPlanImpl
- lritFlagstate 是 **String 型別**（不是 Long）：需要 3 字元國旗代碼或空字串
- ringAlertsFlag 是 **Boolean**："true" 或 "false"（不是 0/1）
- 順序：sbdBundleId → lritFlagstate → ringAlertsFlag

### **錯誤 5: Boolean 值格式錯誤** ⭐⭐

**v6.7 實作（錯誤）**：
```python
def _bool_to_int(self, value: bool) -> str:
    return "1" if value else "0"  # 錯誤！
```

**正確實作**：
```python
def _bool_to_string(self, value: bool) -> str:
    return "true" if value else "false"  # 正確
```

**說明**：
- IWS API 的 Boolean 型別使用 **"true"/"false"** 字串
- **不是** "1"/"0" 或 1/0 數字

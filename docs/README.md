# SBD 管理系統

衛星寬頻數據 (Satellite Broadband Data) 管理系統

## 功能特色

### 客戶介面
- 📱 發起 SBD 啟用申請
- 💰 即時費用計算
- 📋 查詢申請記錄

### 助理介面
- 👨‍💼 財務核准工作台
- ✅ 確認收款並執行 IWS
- 📊 已處理請求查詢

### 共通功能
- 📈 通訊紀錄監視器（台北時間顯示）
- 🔐 角色切換（客戶/助理）

## 快速開始

### 1. 安裝相依套件
```bash
pip install -r requirements.txt
```

### 2. 執行應用程式
```bash
streamlit run app.py
```

### 3. 開啟瀏覽器
系統會自動開啟瀏覽器，或手動前往 http://localhost:8501

## 使用說明

### 客戶申請流程
1. 在側邊欄選擇「客戶 (Customer)」角色
2. 輸入 15 位數 IMEI 號碼
3. 選擇資費方案
4. 查看即時計算的應付金額
5. 點擊「提交申請」

### 助理核准流程
1. 在側邊欄選擇「助理 (Assistant)」角色
2. 查看待核准請求列表
3. 點擊「確認收款並執行 IWS」進行核准

## 🔧 5 大關鍵修正

### 1. Content-Type Action（只有方法名）

```
修正前: action="http://www.iridium.com/activateSubscriber"  ❌
修正後: action="activateSubscriber"                         ✅
```

### 2. XML 命名空間（tns 前綴）

```xml
<!-- 修正前（錯誤） -->
<activateSubscriber xmlns="http://www.iridium.com">

<!-- 修正後（正確） -->
<tns:activateSubscriber xmlns:tns="http://www.iridium.com">
```

### 3. 子元素（無前綴）

```xml
<!-- 正確 -->
<request>
    <iwsUsername>...</iwsUsername>
    <sbdSubscriberAccount>...</sbdSubscriberAccount>
</request>

<!-- 錯誤 -->
<tns:request>  ❌
<request xmlns="...">  ❌
```

### 4. 元素順序（新增 serviceProviderAccountNumber）

```xml
<request>
    <iwsUsername>...</iwsUsername>
    <signature>...</signature>
    <serviceProviderAccountNumber>SP12345</serviceProviderAccountNumber>  ← 新增
    <timestamp>...</timestamp>
    <caller>...</caller>
    ...
</request>
```

### 5. 補全元素

```xml
<plan>
    <sbdBundleId>SBD12</sbdBundleId>
    <lritFlagstate></lritFlagstate>           ← 新增
    <ringAlertsFlag>false</ringAlertsFlag>    ← 新增
</plan>

<deliveryDetail>
    <destination>0.0.0.0</destination>
    <deliveryMethod>DIRECT_IP</deliveryMethod>
    <geoDataFlag>false</geoDataFlag>          ← 新增
    <moAckFlag>false</moAckFlag>              ← 新增
</deliveryDetail>
```

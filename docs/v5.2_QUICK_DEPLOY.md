## 🔧 2 大關鍵修正

### 1. 時間戳記格式

```python
# 修正前
timestamp = "2025-12-25T06:06:05.123456"  # ❌ 有微秒，無 Z

# 修正後
timestamp = "2025-12-25T06:06:05Z"  # ✅ 無微秒，有 Z
```

### 2. 簽章算法（SHA-256）

```python
# 修正前
signature = password  # ❌ 直接使用密碼

# 修正後
import hashlib
signature_input = f"{password}{timestamp}"  # 無空格
signature = hashlib.sha256(signature_input.encode('utf-8')).hexdigest()
# ✅ SHA-256 hex digest
```

**範例**：
```
Password:  "mypass123"
Timestamp: "2025-12-25T06:06:05Z"
Input:     "mypass1232025-12-25T06:06:05Z"  ← 無空格
Output:    "abc123...def456"  ← 64 個字元的 hex
```

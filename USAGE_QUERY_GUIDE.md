# 📊 使用量查詢說明
## Usage Query Guide

---

## 🔍 問題診斷

### **症狀：使用量統計顯示 0**

```
使用量統計：
* 總用量：0 bytes
* 計費用量：0 bytes
* 訊息數：0 則
* Mailbox Check：0 次
* Registration：0 次
```

---

## 🎯 原因與解決方案

### **原因 1: CDR 數據尚未同步**

**解決方案：**

1. **先同步 CDR 數據**
   - 切換到助理模式
   - 選擇「CDR 同步管理」頁面
   - 執行 FTP 同步到 Google Drive

2. **再查詢帳單**
   - 切換回「帳單查詢（新）」
   - 輸入 IMEI 和月份
   - 系統會自動從 Google Drive 查詢 CDR

---

### **原因 2: Google Drive 未連接**

**解決方案：**

**檢查 Secrets 設定：**

在 `.streamlit/secrets.toml` 或 Streamlit Cloud Secrets 中添加：

```toml
[gcp_service_account]
type = "service_account"
project_id = "your-project-id"
private_key_id = "your-key-id"
private_key = "-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
client_email = "your-service-account@your-project.iam.gserviceaccount.com"
client_id = "your-client-id"
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "https://www.googleapis.com/robot/v1/metadata/x509/..."
```

---

### **原因 3: 該設備本月無使用記錄**

這是正常情況：
- 設備未啟用
- 設備已暫停
- 本月確實沒有傳輸數據

**顯示：**
```
ℹ️ 本月無使用記錄或 CDR 數據尚未同步
```

## 📋 正確的使用流程

### **步驟 1: 設定 Google Drive（一次性）**

```bash
# 1. 在 GCP 創建 Service Account
# 2. 下載 JSON 金鑰
# 3. 添加到 .streamlit/secrets.toml
# 4. 重啟 Streamlit
```

### **步驟 2: 同步 CDR（助理端）**

1. 切換到助理模式
2. 選擇「CDR 同步管理」
3. 點擊「開始同步」
4. 等待同步完成

### **步驟 3: 記錄設備操作（助理端）**

1. 選擇「設備操作管理（新）」
2. 記錄設備啟用：
   - IMEI: `300534066711380`
   - 方案: `SBD12`
   - 日期: `2026-01-01`
3. 如有方案變更或暫停，也要記錄

### **步驟 4: 查詢帳單**

1. 選擇「帳單查詢（新）」
2. 輸入：
   - IMEI: `300534066711380`
   - 年份: `2026`
   - 月份: `1`
3. 點擊「查詢帳單」

**預期結果：**
- ✅ 顯示月租費
- ✅ 顯示使用量（如果有 CDR）
- ✅ 顯示操作記錄
- ✅ 顯示計費備註

## 💡 不同頁面的差異

### **「費用查詢」（舊版）**

- 功能：查詢 CDR 使用量和費用
- 特點：
  - ✅ 自動從 FTP 或 Google Drive 查詢 CDR
  - ✅ 顯示詳細的 CDR 記錄
  - ✅ 計算超量費用
  - ❌ 不包含方案變更邏輯
  - ❌ 不包含暫停/恢復邏輯

### **「帳單查詢（新）」（新版）⭐**

- 功能：查詢完整月帳單
- 特點：
  - ✅ 包含所有新的計費邏輯
  - ✅ 升級當月生效
  - ✅ 降級次月生效
  - ✅ 暫停/恢復雙重收費
  - ✅ 行政手續費
  - ✅ 整合 CDR 使用量（如果有）
  - ✅ 顯示操作歷史和備註

---

## 🔄 兩頁面配合使用

### **推薦流程：**

```
1. 使用「費用查詢」（舊版）
   → 查看詳細的 CDR 記錄
   → 確認數據傳輸量
   → 檢查是否有異常使用

2. 使用「帳單查詢（新）」（新版）
   → 查看完整月帳單
   → 包含所有費用（月租+超量+手續費）
   → 了解計費邏輯和備註
```

## ⚠️ 常見問題

### **Q1: 為什麼「帳單查詢（新）」顯示 0？**

**A:** 需要先同步 CDR 或連接 Google Drive

### **Q2: 沒有 Google Drive 可以用嗎？**

**A:** 可以，但：
- 只顯示月租費和手續費
- 不顯示使用量統計
- 無法計算超量費用

### **Q3: 如何確認 CDR 已同步？**

**A:** 
1. 去「CDR 同步管理」頁面
2. 查看「已同步檔案」列表
3. 確認有該月的檔案

### **Q4: 測試時沒有真實 CDR 怎麼辦？**

**A:** 
1. 只記錄設備操作（啟用、變更、暫停）
2. 帳單會顯示：
   - ✅ 月租費
   - ✅ 暫停管理費
   - ✅ 行政手續費
   - ❌ 使用量（顯示 0）
   - ❌ 超量費用（顯示 0）
